## API 
- GET
  - localhost:3000/album/song_title.mp3
  - localhost:3000/album/cover.jpg
- POST
    - localhost:3000/
        - Parameters:
            - coversRequest
            - albumName
            - updateListening
            - changeSong
            - addToPlaylist
            - returnSong
